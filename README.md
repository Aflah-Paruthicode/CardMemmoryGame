# A Simple Memmory game  

a modern memory card game built with react and type script.   
flip cards, find matching pairs, and try to beat your best time.



### Features

- timer system
- best time saved using local storage
- move counter
- sound effects with mute toggle
- card animations
- animated background
- responsive design  

### Tech Stack

- react
- type script
- Css
- Html5 audio Api

### Setup

1. clone the project
   
   ```bash
   git clone https://github.com/Aflah-Paruthicode/Memory_Game.git
   ```

2. navigate into the project folder

   ```bash
   cd ./Memory_Game
   ```

3. install dependencies

   ```bash
   npm install
   ```

4. start building

   ```bash
   npm run dev
   ```
